url: /dosen/create
informasi: ini form dosen

1. Buat Controller : DosenController
    ```
    php artisan make:controller DosenControllerController
    ```
2. buat view
    ```
    php artisan make:view dosen_create
    ```
3. buat route
4. buat function create
5. modifikasi view
